Веб-сниффер (Рыженков Илья)
Запуск python3 main.py (только Linux)
Параметры:
	-h вывод справки
	-i [interface] - выбор сетевого интерфейса
	-F [filename] - выбор файла для записи пойманных пакетов (в формате .pcap)
	-f [rule string] - добавление одного правила в фильтр,
	   возможно указание исходных и целевых ip-адрессов и портов в формате
	   -f s_ip ip_addr1 d_ip ip_addr2 s_port port1 d_port port2.
	   Пакет будет напечатан, если он полностью удовлетворяет хотя бы одному правилу.
	   (или если правил нет)
